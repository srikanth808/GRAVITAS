# GRAVITAS — Crime Intelligence Platform

GRAVITAS is a modern, AI-powered crime intelligence platform built to streamline incident intake and analysis for law enforcement. 

It allows users to upload raw PDF First Information Reports (FIRs), automatically extracting key details and classifying the crime type. The application features a secure, real-time dashboard that visually tracks crime statistics, severity levels, and critical open cases.

## Tech Stack
- **Frontend**: Next.js 16, React, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Deployment**: Vercel

## Access the Platform
The live production version of this application is automatically deployed and accessible at the Vercel URL configured in your dashboard.
